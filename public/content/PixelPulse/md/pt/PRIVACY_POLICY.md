# Política de Privacidade para o Pixel Pulse

**Última Atualização:** 17 de março de 2026

Bem-vindo ao Pixel Pulse! Esta Política de Privacidade explica como o **fertwbr** ("nós", "nos" ou "nosso") lida com as
informações quando você usa os nossos aplicativos móveis e vestíveis Pixel Pulse (o "Serviço").

O nosso compromisso é com a sua privacidade. O Serviço é projetado para funcionar principalmente por meio do *
*processamento local no seu dispositivo**, para minimizar a transmissão de dados externos sempre que possível.

Esta política também reflete a conformidade com os regulamentos brasileiros de privacidade, incluindo a **Lei Geral de
Proteção de Dados (LGPD)** e a **Lei nº 15.211/2025 ("ECA Digital")**, a qual estabelece proteções adicionais para
menores de idade em ambientes digitais.

## 1. Informações que Processamos

### a) Dados de Áudio (Acesso ao Microfone)

**Finalidade:**

A funcionalidade principal do Serviço exige o acesso ao microfone do seu dispositivo para medir os níveis de pressão
sonora ambiente (decibéis).

**Processamento:**

Os sinais de áudio são processados **em tempo real localmente no seu dispositivo** a fim de calcular os níveis de
decibéis.

O sinal de áudio bruto é descartado imediatamente após a medição ser calculada.

**Nós não armazenamos, gravamos ou transmitimos o áudio bruto para fora do seu dispositivo.**

**Gatilho de Coleta:**

O acesso ao microfone pode estar ativo quando:

1. Você está usando ativamente o aplicativo.
2. Você ativa o **Monitoramento em Segundo Plano** (Background Monitoring) ou as **Sessões ao Vivo** (Live Sessions).
3. A medição está em execução em um dispositivo **Wear OS** conectado.

Uma notificação persistente sempre será exibida quando a medição em segundo plano estiver ativa.

### b) Dados de Sessão e Exposição (Armazenados Localmente)

**Finalidade:**

Para fornecer gráficos, histórico, análise de exposição e percepções de segurança auditiva.

**Dados Armazenados:**

* registros de data e hora (timestamps)
* duração da exposição
* valores calculados de decibéis (mín. / méd. / máx.)
* métricas de dose de exposição
* resumos analíticos derivados das sessões gravadas

**Armazenamento:**

Estas informações são armazenadas **localmente num banco de dados privado no seu dispositivo**.

Elas **não são transmitidas para os servidores operados pelo fertwbr**.

Os usuários podem excluir estes dados a qualquer momento ao usar as ferramentas de gerenciamento de dados no aplicativo.

### c) Sincronização com o Wear OS

Se você usar o aplicativo complementar Wear OS, os dados de medição calculados podem ser sincronizados entre o seu
relógio e o seu telefone.

Esta sincronização ocorre por meio do uso da **Android Wearable Data Layer API**, a qual transfere os dados diretamente
entre os seus dispositivos pessoais via Bluetooth, Wi-Fi ou outros métodos de conectividade local.

O áudio bruto **nunca é transferido** entre os dispositivos.

### d) Rotinas Definidas pelo Usuário

O Serviço pode permitir que os usuários configurem rotinas pessoais, tais como **horários de trabalho ou períodos de
sono**.

Estas configurações ajudam a gerar alertas contextuais e percepções analíticas.

Todas as informações de rotina são armazenadas **apenas localmente no seu dispositivo** e nunca são transmitidas para os
nossos servidores.

### e) Informações de Compra no Aplicativo (Pixel Pulse+)

Os recursos premium são desbloqueados por meio de uma **compra única** processada inteiramente pelo **Faturamento do
Google Play** (Google Play Billing).

Nós não coletamos ou armazenamos os detalhes de pagamento.

Nós recebemos apenas um token de validação de licença do Google Play para confirmar o status da compra.

### f) Feedback, Registros e Dados de Diagnóstico Opcionais

O aplicativo fornece uma **ferramenta opcional de suporte no aplicativo** que permite aos usuários o envio de feedback.

O envio de feedback é **totalmente voluntário**.

Os usuários podem escolher incluir:

* um endereço de e-mail para contato
* feedback escrito
* capturas de tela ou anexos
* informações de diagnóstico opcionais

As informações de diagnóstico podem incluir detalhes técnicos, tais como:

* modelo do dispositivo
* versão do Android
* versão do aplicativo
* configurações de idioma / localidade

Estas informações são usadas exclusivamente para a solução de problemas e para a melhoria do Serviço.

Os usuários podem editar ou remover as informações antes do envio.

## 2. Permissões

O aplicativo pode solicitar as seguintes permissões:

* **Microfone:** Exigida para medir os níveis de som.
* **Notificações:** Usada para alertas de exposição e indicadores de serviço de primeiro plano.
* **Servicio de Primeiro Plano:** Exigida para manter as medições precisas enquanto a tela está desligada.

## 3. Compartilhamento e Divulgação de Informações

Nós **não** vendemos, alugamos ou compartilhamos as informações pessoais com anunciantes.

Como a maior parte dos dados do aplicativo é processada e armazenada localmente no dispositivo do usuário, o fertwbr
geralmente **não tem acesso aos dados de medição do usuário**.

As informações podem ser compartilhadas apenas nas seguintes circunstâncias limitadas:

* quando um usuário envia voluntariamente um feedback de suporte
* quando exigido por lei ou solicitação legal válida

## 4. Segurança, Retenção e Exclusão de Dados

### Segurança

Os dados de exposição locais podem ser protegidos por meio dos mecanismos de criptografia suportados pelo sistema
operacional Android e pelas bibliotecas nativas.

### Retenção

Os dados são retidos no dispositivo do usuário pelo tempo que o usuário escolher mantê-los.

### Integração com o Backup do Sistema

Alguns dispositivos podem incluir automaticamente os dados do aplicativo nos **backups do sistema Android** (por
exemplo, backups do Google ou backups do Google Drive).

Estes backups são **controlados inteiramente pelo sistema operacional do dispositivo e pelas configurações de conta do
usuário**, não pelo fertwbr.

O Pixel Pulse **não acessa, gerencia ou armazena estes backups**.

Os usuários podem controlar o comportamento do backup por meio das configurações do seu dispositivo.

### Exclusão

Os usuários podem excluir os dados de exposição gravados a qualquer momento por meio das ferramentas de gerenciamento de
dados no aplicativo.

A desinstalação do aplicativo também removerá os dados armazenados localmente no dispositivo.

## 5. Aviso de Isenção de Responsabilidade Médica

O Pixel Pulse **não é um dispositivo médico**.

Os dados e as percepções fornecidos são estimativas informativas e não devem ser usados para diagnóstico médico
profissional ou como um substituto para o equipamento de proteção auditiva profissional.

## 6. Privacidade das Crianças

O Serviço não é destinado a crianças menores de **13** anos.

No entanto, o aplicativo pode estar acessível a usuários adolescentes a depender das configurações de distribuição da
plataforma.

De acordo com os regulamentos brasileiros de proteção digital, incluindo o **ECA Digital (Lei nº 15.211/2025)**, o
aplicativo é projetado para minimizar o processamento de dados pessoais e não implementa o rastreamento comportamental
ou sistemas de publicidade.

## 7. Alterações a Esta Política de Privacidade

Nós podemos atualizar esta Política de Privacidade periodicamente.

As alterações significativas serão comunicadas ao atualizar a política nesta página e ao atualizar a data de "Última
Atualização".
